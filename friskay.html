friskay- Cat Food Web Page

Designed and developed an engaging cat food web page, "friskay," focusing on a visually appealing interface with integrated animations. The project covered various aspects, including:

  • Header & Navbar Design: Crafted an intuitive and aesthetic header and navigation bar for seamless user interaction.
  • Components: Utilized Figma's component features to create modular and reusable design elements, ensuring consistency throughout the web page.
  • Auto Layout: Implemented responsive design using Figma's Auto Layout feature, ensuring a consistent and visually pleasing layout across different screen sizes.
  • Hero Section: Developed an eye-catching hero section, leveraging compelling visuals and concise copy to capture users' attention and communicate the brand's essence.
  • Veterinarians Testimonials Section: Designed a testimonials section featuring endorsements from veterinarians, enhancing trust and credibility. Implemented animations to create a dynamic and engaging user experience.
