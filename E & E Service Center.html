E & E Service Center Management System

Developed a comprehensive management system using Scene Builder Software for E & E Service Center, an electronic and electrical items repairing center. The system comprises two interfaces: a user interface for store employees and an admin interface for the store owner.

User Management

  • Admin (store owner) creates user accounts using email and 8+ character passwords.
  • All users can update their passwords.
  • Only the main admin registers new users.
  • Users log in with email and password.

Order Management

  • Capture and store details: Order ID, Customer Name, Email, Contact Number, Item, Category, Description, Date.
  • Return option available.
  • Status transitions: Pending → Processing → Completed → Closed.
  • Additional fees for electrical parts shown on the bill.
  • Final bill with net total and item list displayed and emailed.

Item Management

  • Basic details include category, product name.
  • Users can add new items.

Inventory Management

  • Users can add items under main categories.
  • Update and remove items.
  • Four zones: Orange, Red, Yellow, Green, based on process and completion times.
  • Quantity tracking based on status.

Reports

  • Display Order Reports
  • Display Customer Reports
  • Display Sales Report
